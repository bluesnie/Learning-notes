  - 渲染环境
  - 将动画保存为 GIF
  - 将动画保存为视频
  - 3D 绘图

## Python 脚本与 YAML 配置文件

```python
import irsim

# 在 make() 函数中将 save_ani 设为 True，即可轻松把仿真动画保存为 GIF
env = irsim.make(save_ani=True) # 将动画保存为 GIF
env = irsim.make(projection='3d') # 3D 绘图

for i in range(300):

    env.step()
    env.render(0.05)

    if env.done():
        break

env.end(ending_time=3)
```
