# 投资误区

- 误区一：基金一定要长期投资吗？

    把握时机，不能为了长期投资而长期投资，就好比春种秋收

- 误区二：基金净值是越便宜越好吗？

    - 并非如此
    - 净值低：
        - 新基金：不推荐
        - 业绩差：不推荐
        - 老基金，业绩也不错：通过分红或拆分减下来，不推荐
    - 净值高：业绩优秀
    
- 误区三：是不是分红越多的基金越好？

    - 分红和基金的好坏没有必然关系，有恶性分红来实现二次营销就像上面说的。
