# 什么是ROS?

- 中间件/类操作系统
    - 硬件抽象
    - 底层设备控制
    - 常用函数的实现
    - 进程间消息传递
    - 包管理

- 官方的定义：ROS = 框架 + 工具 + 功能 + 社区

    - 框架
        - 分布式
        - 节点(进程)管理
        - 节点(进程)间通信
    - 工具
        - 仿真
        - 数据可视化
        - 图形界面
        - 数据记录
    - 功能
        - 控制
        - 规划
        - 视觉
        - 建图
    - 社区
        - 软件包管理
        - 文档
        - 教程
